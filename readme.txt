** Art Of Sequence **
*********************

Description :
	*Art Of Sequence* is a *free* and *open-source* set of tools for *Digital Sequence* edition and publication.

Technical : 
	
This repository is set to gather all the related original projects of A.O.S.
It uses Mercurial Sub-Repo feature (see http://mercurial.selenic.com/wiki/Subrepository )
You just need to update this repo to get all the other sub-repos.

Read the .hgsub file to know where the sub-repos are located. 

Our current public repositories are all hosted at :
	http://code.google.com/p/art-of-sequence/

You can see te code of all sub-repos there.
